Modules

Organize code into separate files with export and import.

Understanding Modules

Modules let you split code into separate files and reuse functions/classes across your project.

  • Named Export: export function name() { } - Export multiple things
  • Default Export: export default ClassName - Export one main thing
  • Import Named: import { name } from './file.js'
  • Import Default: import ClassName from './file.js'

Works on Vercel! Dynamic import() works in all modern browsers and on Vercel deployments.

What's in modules.js?

The modules.js file contains:

  • Named exports: calculateArea(), calculatePerimeter(), isSquare()
  • Default export: Rectangle class
  • Bonus: PI constant, Circle class, and calculateCircleArea()
View modules.js code 
// Named exports
export function calculateArea(width, height) {
    return width * height;
}

export function calculatePerimeter(width, height) {
    return 2 * (width + height);
}

export function isSquare(width, height) {
    return width === height;
}

// Default export
export default class Rectangle {
    constructor(width, height) {
        this.width = width;
        this.height = height;
    }
    getArea() { return this.width * this.height; }
    getPerimeter() { return 2 * (this.width + this.height); }
}

// More named exports
export const PI = 3.14159;
export function calculateCircleArea(radius) {
    return PI * radius * radius;
}
export class Circle {
    constructor(radius) { this.radius = radius; }
    getArea() { return calculateCircleArea(this.radius); }
}

Try These Examples

1. Import specific functions:

import { calculateArea, calculatePerimeter } from './modules.js';
console.log(calculateArea(3, 4)); // 12

2. Import with alias:

import { calculateArea as getArea } from './modules.js';
console.log(getArea(5, 6)); // 30

3. Import default export:

import Rectangle from './modules.js';
const rect = new Rectangle(10, 20);
console.log(rect.getArea()); // 200

4. Import everything:

import * as MathUtils from './modules.js';
console.log(MathUtils.calculateArea(7, 8)); // 56
const rect = new MathUtils.default(5, 5);

Note: Dynamic import() works everywhere:

  • Works on Vercel (production deployments)
  • Works locally (with or without a server)
  • Works in all modern browsers
  • For static import statements, use <script type="module">
